# Aegis Institute Website

Static frontend website built for the MANTRA 2026  Assignment 1.

## Features
- Full semantic structure with clean HTML5 layouts.
- Completely responsive grid systems targeting mobile, tablet, and desktop viewports.
- Interactive mobile drawer navigation toggle, accordion FAQ panels, and regular expression client-side form validation.
- Implements individual title, meta description tags, heading structures, and comprehensive alt descriptors.
