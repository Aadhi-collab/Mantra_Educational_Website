# Aegis Institute of Technology & Data Science Website

A complete responsive frontend static website developed fully compliant with the **MANTRA 2026 Summer School Assignment 1** guidelines.

## 🌟 Assignment Rules Check
- **HTML5 Elements:** Proper page structure, navigation links, layout wrappers.
- **CSS3 Styling & Responsiveness:** Fully fluid desktop, tablet, and smartphone compatibility utilizing grid configurations and targeted media queries.
- **JavaScript Enhancements:** Built-in mobile drop-down toggle navigation, fully fluid accordion FAQ drawers, and automated client-side regular expression form validation layout.
- **Comprehensive SEO Architecture:** Every singular file contains distinct standalone `<title>`, meta description tags, targeted search keywords, strict hierarchy headings (`<h1>` used exactly once per view, segmented via `<h2>` and `<h3>`), and highly descriptive image `alt` text.

## 📂 Project Structure
- `index.html` - Home Page (Banners, call-to-actions, highlight tiles, FAQ system)
- `about.html` - About Page (Overview, core philosophy, faculty timetables schedule matrix)
- `courses.html` - Offerings View (Course dynamic details layout cards, mock download materials handles)
- `gallery.html` - Showcase View (Optimized asset displays with caption layers and semantic alt bindings)
- `contact.html` - Portal Page (HQ communication coordinates alongside secure client-validated admission ledger framework)
- `style.css` - Custom styling stylesheet containing global presets and custom media query view handlers
- `script.js` - Lightweight interactivity runtime
